# Infrastructure (Terraform)

Provisions the full production stack for Parent Check on AWS:

```
Internet
  → ALB (:80)                       aws_lb + listener + target group
  → ECS Fargate service             aws_ecs_service (task: gunicorn on :8000)
      ├─ RDS PostgreSQL 16 (pgvector)   aws_db_instance
      ├─ ElastiCache Redis 7            aws_elasticache_cluster
      └─ Secrets Manager                SECRET_KEY / DATABASE_URL / REDIS_URL / GEMINI_API_KEY
  + ECR repository                   image registry (deploy.yml pushes here)
  + IAM                              task roles + GitHub OIDC deploy role
```

Security groups chain the tiers: only the ALB reaches the app, only the app
reaches Postgres/Redis. Credentials are generated by Terraform, stored in
Secrets Manager, and injected into the task as env vars — never written to the
task definition or to CI.

## Usage

```bash
cd infra
terraform init
terraform plan  -var="gemini_api_key=..."   # review
terraform apply -var="gemini_api_key=..."
```

Then wire CI/CD to what Terraform created:

```bash
terraform output ecr_repository_url        # -> ECR_REPO in deploy.yml
terraform output github_deploy_role_arn    # -> ROLE_ARN in deploy.yml
terraform output app_url                   # public URL
```

## Notes / production hardening

- **Not `terraform apply`-tested here** (no AWS account in this environment).
  The config is written to plan/apply cleanly; run `terraform validate` and a
  `plan` against your account before relying on it.
- Uses the **default VPC** for brevity. For production, create a dedicated VPC
  with private subnets for RDS/Redis and keep only the ALB public.
- The **GitHub OIDC provider** is referenced as an existing data source. If your
  account doesn't have it yet, add an `aws_iam_openid_connect_provider` resource
  for `token.actions.githubusercontent.com`.
- `skip_final_snapshot` and `recovery_window_in_days = 0` are demo conveniences
  — remove them for real data.
- Add HTTPS: an ACM cert + a `:443` listener, and redirect `:80 → :443`.
